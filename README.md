| ----   |:---:|
| Up     | W   |

| Left   | A   |

| Start  | T   |
| Select | Y   |
| A      | G   |
| B      | H   |


## NPM Commands
- start -- Creates HTTP Server delivering ./target at 127.0.0.1:8080
- clean 
- build -- Generates the output artifact
- package -- Packages the output artifact
